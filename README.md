# JalanAman 

Aplikasi pelaporan jalan rusak, lampu lalu lintas mati, dan potensi bahaya lalu lintas lainnya.

## Fitur
- Lapor kondisi jalan dalam satu klik
- Lihat laporan pengguna lain
- Peta lokasi laporan
- Simpan data ke Firebase
